## When Does AI Intervene?

### AI is NOT used when:
- User has disabled "Enhanced Mode" in settings
- Text is very short (< 15 characters)
- Model hasn't finished loading yet
- WebGPU is not supported in the browser
- Text contains no homophones or context-dependent words

### AI IS used when (any of these triggers):
1. **Text contains homophones** - ALWAYS triggers AI for context analysis:
   - `their` / `there` / `they're`
   - `your` / `you're`
   - `its` / `it's`
   - `to` / `too` / `two`
   - `then` / `than`
   - `affect` / `effect`
   - `lose` / `loose`

2. **Text is longer than 100 characters** - benefits from context

3. **Text contains capitalized words mid-sentence** - potential proper names

### What AI does differently:

**Without AI (Local Dictionary):**
- Flags ALL homophones as yellow "choose meaning" highlights
- User must manually decide which is correct
- Example: "your" flagged with suggestions "you're"

**With AI (Enhanced Mode):**
- AI reads the full sentence to understand context
- Only flags the word if it's WRONG in that context
- Example: "Your going to the store" â†’ AI detects "Your" should be "You're"
- Example: "Your bag is here" â†’ AI sees this is CORRECT, no highlight

## Privacy

- **Local Dictionary**: 100% offline, no network requests
- **Browser AI**: Runs entirely in browser via WebGPU, text never sent to servers
- **Storage**: All settings stored in `chrome.storage.local` (never synced)
- **No Telemetry**: Zero analytics or tracking
